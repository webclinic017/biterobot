Strategy writing

1

Create your strategy:

Create file with .py extension and write your strategy inside
or
Write your strategy right in the editor in Strategy editor tab
2

Check if:

  • You imported class backtrader.Strategy
  • Your strategy class named Strategy
  • Class Strategy is inherited from backtrader.Strategy
  • All the methods of your Strategy class you might need to use are redefined (see strategy docs for more information)
3

Upload your strategy:

  • Hit the “Add” button
  • Upload your .py file or write your code right inside editor
  • Hit the “Submit” button
Congratulations! Your strategy is ready for the backtesting!
